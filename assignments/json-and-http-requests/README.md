# 📘 Assignment: JSON and HTTP Requests

## 🎯 Objective

Learn how to use Python to request data from a public API, parse JSON responses, and present the information in a simple program.

## 📝 Tasks

### 🛠️ Request Data from an API

#### Description
Use Python to make a request to a public API and inspect the response.

#### Requirements
Completed program should:

- Use the built-in urllib module or another standard library approach to request data
- Handle the response as JSON
- Print or display the received data clearly

### 🛠️ Extract Useful Information

#### Description
Turn the raw API response into a simple, readable summary.

#### Requirements
Completed program should:

- Extract at least one relevant field from the JSON data
- Format the output in a student-friendly way
- Demonstrate how to work with nested data structures if needed

### 🛠️ Build a Small CLI Program

#### Description
Create a small command-line program that shows what the student learned.

#### Requirements
Completed program should:

- Accept a simple input or use a fixed example
- Display a short report based on the API data
- Include clear comments or labels in the output
